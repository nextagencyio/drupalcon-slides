# DrupalCon Slides

Video slide deck built with Vite for DrupalCon presentations.

## Slides

1. **Decoupled vs Monolith** — fighter video
2. **Install, Import & Deploy to Vercel**
3. **Connect to Vercel**
4. **Import content with Chatbot**
5. **One-shot coffee shop**

Each section has a title card (white text on black) followed by an auto-playing muted video.

## Features

- Videos autoplay muted and advance to the next slide on completion
- Title slides auto-advance after 3 seconds
- Keyboard navigation: arrow keys, spacebar
- Touch swipe support
- Click/tap to advance
- URL hash bookmarks (`#slide1`, `#slide2`, etc.)
- Loops back to the beginning after the last slide

## Development

```bash
npm install
npm run dev
```

Deployed on Vercel with auto-deploy from GitHub.
